# Muuday Documentation Map

Documentacao operacional e de estrategia organizada por uso:

- `docs/handoff/`: handoff entre assistentes/agentes.
- `docs/reports/`: relatorios e auditorias.
- `docs/roadmaps/`: roadmaps de execucao.
- `docs/setup/`: guias de setup sem risco.
- `docs/planning/`: backlog e planejamento tecnico.
- `docs/operations/`: runbooks de release/incidente/rollback.
- `docs/automations/`: contratos e blueprints de automacao.
- `docs/crm/`: materiais de CRM/HubSpot.
