# Operations Docs

This folder contains runtime-safe operational documentation.

## Files

- `RELEASE_CHECKLIST.md`: pre/post-release checks.
- `ROLLBACK_RUNBOOK.md`: rollback procedure for app and database.
- `INCIDENT_RUNBOOK.md`: response flow for production incidents.

## Usage

1. Follow the release checklist before every production deploy.
2. If an incident occurs, open an incident issue and use the incident runbook.
3. If rollback is required, use the rollback runbook and record timeline/actions.
